Aspose.LLM for .NET is an on-premise API for integrating large language model capabilities into .NET applications. It lets developers create chat-driven workflows, use built-in presets for supported models, manage sessions, persist conversations, and customize runtime behavior for local or controlled deployments.

## Integrate LLM Capabilities in .NET Applications

> ### Chat Sessions
> Build interactive conversational workflows with single-turn and multi-turn chat scenarios.

> ### Model Presets
> Start quickly with built-in presets for supported models and adjust parameters for your own scenarios.

> ### Session Persistence
> Save and restore chat state to continue conversations across runs or user sessions.

> ### Multimodal Input
> Send text prompts together with optional image input where supported by the selected model preset.
