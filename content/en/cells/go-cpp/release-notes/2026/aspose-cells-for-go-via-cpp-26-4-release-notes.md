## **Public API and Backwards Incompatible Changes**

The following is a list of any changes made to the public API such as added, renamed, removed or deprecated members as well as any non-backward compatible change made to Aspose.Cells for Go via C++. If you have concerns about any change listed, please raise it on the Aspose.Cells support forum.

### **Adds two methods in PivotOptions class.**

Supports to get or set a value indicating whether to show expand/collapse field buttons.

{{% alert color="primary" %}}

func (instance *PivotOptions) GetShowExpandCollapseFieldButtons() (bool, error)
func (instance*PivotOptions) SetShowExpandCollapseFieldButtons(value bool) error

{{% /alert %}}

### **Adds and obsoletes some methods in Picture class.**

Add:

Indicates whether to place the image in a cell or over cells.

{{% alert color="primary" %}}

func (instance *Picture) IsPlacedInCell() (bool, error)
func (instance*Picture) SetIsPlacedInCell(value bool) error

{{% /alert %}}

Obsoletes:

Please use IsPlacedInCell()/SetIsPlacedInCell() method instead.

{{% alert color="primary" %}}

func (instance \*Picture) PlaceInCell() error

{{% /alert %}}

### **Adds some methods in NumbersLoadOptions class.**

Indicates whether to preserve table names when importing from Numbers.

{{% alert color="primary" %}}

func (instance *NumbersLoadOptions) GetPreserveTableName() ( bool, error )
func (instance*NumbersLoadOptions) SetPreserveTableName() error
func (instance *NumbersLoadOptions) GetLoadTableType() ( LoadNumbersTableType, error )
func (instance*NumbersLoadOptions) SetLoadTableType(value LoadNumbersTableType) error

{{% /alert %}}

### **Adds new enum member OpenDocumentFormatVersionType::Odf14.**

Support ODF Version 1.4.

### **Adds some methods in PivotTable class.**

Gets or sets the table style settings of this pivot table.

{{% alert color="primary" %}}

TableStyle GetPivotTableStyle()
func (instance *PivotTable) GetPivotTableStyle() (*TableStyle, error)
func (instance *PivotTable) SetPivotTableStyle(value*TableStyle) error

{{% /alert %}}

Represents the blank area at the top-right of the PivotTable (top-left for RTL sheets).

{{% alert color="primary" %}}

func (instance *PivotTable) GetTopRightArea() (*CellArea, error)

{{% /alert %}}

Gets the region of the filter area.

{{% alert color="primary" %}}

func (instance *PivotTable) GetFilterArea() (*CellArea, error)

{{% /alert %}}

Gets the button area for a specified axis type.

{{% alert color="primary" %}}

func (instance *PivotTable) GetButtonArea( axisType*PivotFieldType ) (\*CellArea, error)

{{% /alert %}}

### **Adds new method Worksheet.GetAllPictures().**

Gets all pictures including images embedded in cells and over the cells.

{{% alert color="primary" %}}

func (instance *Worksheet) GetAllPictures( ) (*Picture[], error)

{{% /alert %}}

### **Adds two methods in HtmlLoadOptions class.**

Specifies how &lt;p&gt; elements are rendered when loading HTML.

{{% alert color="primary" %}}

func (instance *HtmlLoadOptions) GetParagrahLayoutMode( ) (HtmlParagraphLayoutMode, error)
func (instance*HtmlLoadOptions) SetParagrahLayoutMode( value HtmlParagraphLayoutMode ) error

{{% /alert %}}

### **Obsoletes two methods in ChartFrame class.**

Please use ChartFrame.GetArea().GetFillFormat().GetFillType() / ChartFrame.GetArea().GetFillFormat().SetFillType(FillType::Automatic) instead.

{{% alert color="primary" %}}

func (instance *ChartFrame) GetBackgroundMode( ) (BackgroundMode, error)
func (instance*ChartFrame) SetBackgroundMode( value BackgroundMode ) error

{{% /alert %}}

### **Adds new enum HtmlParagraphLayoutMode.**

Represents how &lt;p&gt; elements are rendered during HTML load.

### **Adds two methods in TableStyle class.**

Creates a TableStyle instance with the specified name for the given worksheets collection.

{{% alert color="primary" %}}

func (instance *TableStyle) Create(name string, sheets*WorksheetCollection ) (TableStyle, error)

{{% /alert %}}

### **Adds two members in TableStyleElementType enum.**

Represents a style element that applies to the header of a pivot table’s grand total column.

{{% alert color="primary" %}}

TableStyleElementType_GrandTotalColumnHeader

{{% /alert %}}

Represents a style element that applies to the header of a pivot table’s grand total row.

{{% alert color="primary" %}}

TableStyleElementType_GrandTotalRowHeader

{{% /alert %}}

### **Adds two methods in CopyOptions class.**

Indicates whether to refer to target worksheet if the referred worksheet is not copied.

{{% alert color="primary" %}}

func (instance *TCopyOptions) GetUpdateInvalidReferencesToTargetSheet() (bool, error)
func (instance*TCopyOptions) SetUpdateInvalidReferencesToTargetSheet(value bool) error

{{% /alert %}}

### **Adds three members in ErrorCellValueType enum.**

Represents the value of a cell containing a #VALUE! error.

{{% alert color="primary" %}}

ErrorCellValueType_Value

{{% /alert %}}

Represents the value of a cell containing a #REF! error.

{{% alert color="primary" %}}

ErrorCellValueType_Ref

{{% /alert %}}

Represents the value of a cell containing a #NAME? error.

{{% alert color="primary" %}}

ErrorCellValueType_NA

{{% /alert %}}
